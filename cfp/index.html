Call For Proposals Results

CFP Results are Here!

We are pleased to announce the CFP results. We have received 51 submissions before our deadline 2017/02/28 23:00 (GMT+8). The volunteers in our Program Committee had a hard time to shortlist them to fit in our agenda.

Here is the list of the accepted CFP topics, along with some invited sessions & confirmed sponsor sessions:

  • Gandi's LiveDNS and new gTLD's (Thomas Kuiper) (Gandi)
  • The deep dive in-and-out of MySQL Group Replication (Ivan Ma) (MySQL)
  • What's New in MySQL 8.0 (Ryusuke Kajiyama) (MySQL)
  • Preparing for the App-ocalypse: The Web in the Age of Physical Computing (Dietrich Ayala) (Mozilla)
  • Reproducible builds: Fulfilling the original promise of free software (Chris Lamb) (Debian)
  • Building a career in Open Source (Vaishali Thakkar) (India)
  • Open CV in Raspberry PI with Camera (Kazuhiko Inaba) (Japan)
  • Tips in setting up OpenStack from scratch (Vinnson Lee)
  • Let's Rust! System programming is awesome again! (Karl Lam) (OneSky)
  • One Year of Rust Microkernel Development (Wei Tang)
  • Building Artificial Intelligence Units in Rust (Vigneshwer Dhinakaran) (India)
  • Data In, Elasticsearch Out (Joshua Rich) (Elasticsearch)
  • A War of File Format -- Why We Select ODF in Taiwan (Franklin Weng) (Taiwan)
  • PIME - 用 Python 和 JavaScript 快速開發 Windows 的中文輸入法 (PC Man) (Taiwan)
  • ezgo for HK：自由的學習系統，本地的楷書字型。 (Franklin Weng, Kingman Leung)
  • Free Hong Kong Kai Font (Kingman Leung)
  • This year in #rstats (Hong Kong R User Group)
  • Harnessing social media with R (Lushi Chen)
  • A Security State of Mind: Continuous Security for DevOps with Kubernetes (Chris Van Tuin) (Redhat)
  • Build a cross-platform game in Haxe (Andy Li, Kevin Leung) (Workshop) (Haxe Foundation)
  • Getting into the virtual world with WebVR/A-Frame (Andy Li, Parvani Ng) (Workshop)
  • Building web app with open source serverless technology (David Ng) (Workshop) (Oursky)
  • 1000 days of Mozilla Community Space in Taipei (Irvin Chen) (Taiwan)
  • Blender Interface and 3D-CGI concepts (Ann/y Leung)
  • 軟體訊息本地化前置翻譯工具 (趙惟倫) (Taiwan)
  • ZMQ multiplex between golang coroutine and python thread (Rick Mak) (Oursky)
  • Pentest fun with metasploit framework II (Hayden Chung)
  • Waiting too long for Excel's VLOOKUP? Use SQLite for simple data analysis! (Amanda Lam) (Workshop)
  • AWS Cloud Lab (革新雲端電腦房) (Cyrus Wong)
  • Why BASH? (Daniel Chcouri) (MeteorSpark)
  • 沒有 Powerpoint 的簡報世界 (Franklin Weng) (Taiwan)
  • JavaScript fuzzing in Mozilla, 2017 (Gary Kwong) (Mozilla)

If your topic is not selected, don't worry. You may still share your topic in our Unconference Session at of Day 2 (Saturday).
