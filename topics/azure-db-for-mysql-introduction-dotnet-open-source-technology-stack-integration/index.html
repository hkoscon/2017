Talk

Azure DB for MySQL introduction and.NET Core open source technology stack integration

Edwin Song

After a brief introduction of MySQL PaaS service, we will introduce the integration with .NET Core stack. MySQL's traditional technology stack is a complete ORM framework based on Linux's LNMP/LAMP and Java. With .NET Core advantages on both open source and cross platform, how can the.NET stack such as ORM (Entity Framework) be better supported with MySQL? With two domestic open source projects mysqlconnector Ado.net provider) and Pomelo (EF MySQL Provider), we could enjoy the convenience of the full support on MySQL with .NET Core.

Day 2 (10th June) 10:00 AM - 10:30 AM
